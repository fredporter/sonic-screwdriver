
# Repo Boundary

Sonic is responsible for deployment and hardware bootstrap.

It does not own runtime contracts.

## Sonic owns

deployment planning  
USB provisioning  
manifest generation  
device catalog  

## Sonic does not own

uDOS runtime semantics  
uHOME runtime contracts  
Wizard networking or API orchestration  

Those belong to:

uDOS-core  
uDOS-wizard  
uHOME-server
