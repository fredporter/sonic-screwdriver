# Docs Compost

`docs/.compost/` is the tracked archive for superseded Sonic documentation.

Use it for:

- legacy lesson chains
- historical exploration briefs
- older roadmap material
- superseded specs kept only for reference
- retired course scaffolds that are no longer part of the active learner path

Active docs stay in `docs/`, `wiki/`, and `courses/`.

This folder is intentionally removed from active indexes and learner-facing
navigation.
