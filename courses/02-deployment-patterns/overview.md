# Course Overview

Course 02 focuses on deployment strategy, not core command syntax.

Where Course 01 teaches `plan -> inspect -> dry-run -> apply -> hand off` for a
single deployment, Course 02 teaches how to scale that model across multiple
machines and multiple rollout waves.

Core progression:

`single host safety -> staged rollout -> fleet repeatability -> post-deploy audit`

This course keeps Sonic boundaries explicit. Runtime orchestration and service
ownership still belong to sibling systems where appropriate.
