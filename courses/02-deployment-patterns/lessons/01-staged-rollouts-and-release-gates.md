# Lesson 01 - Staged Rollouts and Release Gates

Status: Outline stub

## Why This Lesson Exists

Staged rollout reduces blast radius. Instead of applying to all targets at once,
we advance in small waves gated by validation evidence.

## Planned Sections

- rollout wave design (`canary -> pilot -> broad rollout`)
- release-gate checklist before each wave
- stop conditions and rollback triggers
- communication and handoff protocol between waves

## Planned Hands-On

- define objective pass/fail gates for each wave
