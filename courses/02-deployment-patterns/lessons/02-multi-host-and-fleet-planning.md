# Lesson 02 - Multi-Host and Fleet Planning

Status: Outline stub

## Why This Lesson Exists

Fleet planning introduces host classes, hardware variance, and scheduling
constraints that are not visible in a single-host lesson.

## Planned Sections

- host classification and compatibility grouping
- manifest strategy for shared vs host-specific values
- scheduling patterns for maintenance windows
- fleet readiness checklist and preflight reporting

## Planned Hands-On

- split a sample fleet into compatibility groups
- draft one shared checklist and one host-class checklist
- design a staged maintenance schedule for minimal downtime

## Planned Outputs

- `fleet-groups.csv`
- `fleet-checklist.md`
- `maintenance-window-plan.md`
