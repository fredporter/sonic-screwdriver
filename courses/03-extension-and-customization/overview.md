# Course Overview

Course 03 shifts from operation to extension design.

The core question is not "can this run" but "can this evolve safely over time"
across public APIs, service contracts, and packaging expectations.

Core progression:

`identify extension point -> implement minimal customization -> verify contracts -> package responsibly`

This course reinforces family boundaries. Sonic extension work should remain in
Sonic-owned surfaces and integrate cleanly with sibling systems.
