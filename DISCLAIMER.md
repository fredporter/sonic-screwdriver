# Disclaimer

`sonic-screwdriver` plans and applies storage, partitioning, installer, and
recovery workflows. Running these flows against the wrong device can destroy
data or render a system unbootable.

Before using this repository on real hardware:

- validate manifests and dry runs first
- confirm target device identifiers and partition layouts
- verify the license and redistribution status of third-party payloads
- test on disposable hardware when introducing new flows
