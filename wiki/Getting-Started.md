# Getting Started

If you are new to Sonic, use this order:

1. read [../README.md](../README.md)
2. read [Repo Map](Repo-Map.md)
3. start [../courses/01-sonic-screwdriver/README.md](../courses/01-sonic-screwdriver/README.md)
4. review [../docs/v1/howto/dry-run.md](../docs/v1/howto/dry-run.md)
5. only then inspect real apply scripts

For broader learning beyond Sonic itself, continue into the wider uDOS v2
family docs, especially `uDOS-docs`, `uDOS-core`, and `uDOS-wizard`.

## First Practical Habit

Always separate:

- planning
- inspection
- destructive execution

That separation is the core safety and learning habit in this repo.
