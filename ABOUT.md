# About

`sonic-screwdriver` is the maintained installer, distributor, and deployment
surface for the free public uDOS release.

It is the operational companion to `sonic-ventoy`:

- `sonic-ventoy` owns boot templates, Ventoy-facing configuration, and media
  structure defaults
- `sonic-screwdriver` owns planning, verification, payload staging, recovery
  flows, and device-apply execution

This repository is maintained separately from the older `uDOS-family`
collection, but it still points broader family reading toward the public GitHub
Pages family library index instead of copying every family-wide document into
the repo.

Public family docs:

- repo: `https://github.com/fredporter/uDOS-docs/`
- Pages index: `https://fredporter.github.io/uDOS-docs/`
